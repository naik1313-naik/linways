# Linways 3D Portfolio (Astro)

Modern premium portfolio website built with:

- **Astro** (main framework)
- **React** components inside Astro
- **Three.js** with `@react-three/fiber` + `@react-three/drei`
- **Tailwind CSS** styling
- **GSAP** intro + scroll animations

## Features

- Full-screen interactive 3D hero scene
- Sticky navbar with smooth anchor scrolling
- Animated About section
- 3D-style project cards with hover tilt
- Kudos section with **localStorage** persistence
- Contact form UI
- Dark, glassmorphism, responsive design

## Installation

```bash
npm install
```

## Run in development

```bash
npm run dev
```

Open: `http://localhost:4321`

## Build for production

```bash
npm run build
```

## Optional type checks

```bash
npm run check
```
