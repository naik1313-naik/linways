# linways

Modern 3D portfolio website built with **Astro**, **React**, **Three.js**, **Tailwind CSS**, and **GSAP**.

## Features

- Full-screen 3D hero section with animated object and scroll-reactive camera movement
- Sticky glassmorphism navbar with smooth scroll navigation
- About section with modern UI and reveal animation
- 3D-style project cards with interactive hover tilt
- Kudos section with localStorage-backed messages (name + message)
- Contact section UI for quick outreach
- GSAP intro and scroll-triggered transitions
- Dark, gradient-rich, responsive design

## Installation

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:4321`).

## Build for production

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```
